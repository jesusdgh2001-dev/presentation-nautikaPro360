## Publicar en GitHub Pages

1. Sube todos los archivos a tu repositorio en la raíz
2. Ve a **Settings → Pages**
3. En "Source" selecciona **main branch / root**
4. Guarda — en ~2 min tu página estará en `https://TU-USUARIO.github.io/TU-REPO`
